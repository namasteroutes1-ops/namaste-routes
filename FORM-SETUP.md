# Form Data Kaise Milega? (Hindi Guide)

## Abhi data kyun nahi aa raha?

Aap website **local computer** se khol rahe ho (`file://`). Is case mein:
- **Netlify Forms** kaam nahi karta (site live nahi hai)
- **Email** ke liye ek baar setup chahiye (neeche)

---

## ✅ Sabse Aasaan — Web3Forms (2 minute, FREE)

Yeh sabse reliable tareeka hai — har inquiry **seedha Gmail** par aayegi.

### Steps:
1. [web3forms.com](https://web3forms.com) kholo
2. Email: `namaste.routes.1@gmail.com`
3. **Create Access Key** click karo
4. Jo key mile, copy karo
5. File kholo: `js/config.js`
6. Paste karo:
   ```js
   WEB3FORMS_KEY: "apni-key-yahan-paste-karo",
   ```
7. Save karo, browser refresh karo
8. Form test karo → **Gmail inbox check karo**

---

## ✅ Option 2 — FormSubmit Email Activate

Agar Web3Forms nahi use karna:
1. Form ek baar submit karo
2. Gmail `namaste.routes.1@gmail.com` kholo
3. **Spam/Promotions** folder bhi dekho
4. **FormSubmit** se email aayegi — **Activate/Confirm** link click karo
5. Uske baad har inquiry email par aayegi

---

## ✅ Option 3 — Netlify (website live hone ke baad)

Jab site Netlify par deploy ho:
1. [netlify.com](https://netlify.com) → apni site
2. **Forms** tab → saari inquiries yahan dikhengi
3. Email notification: Site settings → Forms → Notifications
